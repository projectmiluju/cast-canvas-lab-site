# CastCanvas Lab — MVP Scope

## MVP 목표

넓은 캔버스 위에 문서와 이미지를 자유롭게 배치하고 정리할 수 있는 무드보드형 워크스페이스.
로그인한 사용자는 다른 사람과 실시간으로 함께 편집할 수 있다.

---

## 사용자 유형

| 유형            | 설명                                              |
| --------------- | ------------------------------------------------- |
| 비로그인 사용자 | 로컬 전용 캔버스 사용 가능 (저장 불가, 협업 불가) |
| 로그인 사용자   | 워크스페이스 생성, 저장, 실시간 협업 사용 가능    |

---

## MVP 범위

### 캔버스

- 무한 캔버스 위에서 팬(pan) / 줌(zoom)
- PDF 파일을 캔버스에 드롭 → 문서 노드로 배치 (캔버스 인라인 뷰)
- 문서 노드 선택 시 사이드 패널에서 PDF 상세 열람 가능
- 이미지 파일을 캔버스에 드롭 → 이미지 노드로 배치
- 노드 이동, 크기 조정
- 노트 노드 생성 및 텍스트 입력
- 노드 간 연결선(엣지) 생성 및 삭제
- 노드 삭제

### 인증

- 이메일/비밀번호 회원가입 및 로그인
- 비로그인 상태: 로컬 전용 캔버스 사용 가능 (저장 불가, 협업 불가)
- 로그인 상태: 전체 기능 사용 가능

### 워크스페이스

- 워크스페이스 생성
- 워크스페이스당 캔버스 2~3개 생성 가능
- 캔버스 저장 및 불러오기
- 워크스페이스 공유 링크 생성 (협업 초대용)

### 실시간 협업

- 같은 워크스페이스에 접속한 사용자 간 캔버스 실시간 공동 편집
- 다른 사용자의 커서/존재 표시 (awareness)
- 협업은 로그인 필수

### 파일 업로드

- PDF, 이미지(PNG/JPG) 드롭 업로드 지원
- 파일은 오브젝트 스토리지에 저장

---

## MVP 제외 항목

- 문서 내 텍스트 검색 및 인덱싱
- 캔버스 버전 히스토리
- 댓글 / 스레드
- 템플릿
- 모바일 지원
- 워크스페이스 멤버 권한 관리 (초대자 = 편집자로 단순화)
- 알림

---

## 레포별 MVP 담당 범위

### `cast-canvas-lab-fe`

- 무한 캔버스 UI (React Flow 기반)
- 문서/이미지/노트 노드 컴포넌트
- 노드/엣지 상호작용 (드래그, 리사이즈, 연결, 삭제)
- 회원가입 / 로그인 페이지
- 워크스페이스 생성 및 진입
- Yjs 협업 클라이언트 연동
- 협업 사용자 커서 표시

### `cast-canvas-lab-be`

- 회원가입 / 로그인 / 인증 토큰 발급
- 워크스페이스 CRUD
- 캔버스 메타데이터 저장 (노드/엣지 위치, 타입 등)
- 파일 업로드용 서명된 URL 발급
- 업로드 완료 후 에셋 메타데이터 저장
- 협업 접근 권한 검증 (collab 서버 연동)

### `cast-canvas-lab-collab`

- Yjs WebSocket 서버
- 워크스페이스 세션 관리 (입장 / 퇴장)
- 실시간 캔버스 상태 동기화
- awareness/presence 전파
- 협업 접근 시 백엔드 인증 검증

---

## 비로그인 접근 정책

- 앱 진입 시 로컬 전용 캔버스 바로 사용 가능
- 파일 드롭, 노드 배치, 노트 작성 등 캔버스 편집 가능
- 저장, 워크스페이스, 협업 기능은 불가
- 로그인 시 로컬 캔버스 내용 유지 여부는 미정 (Open Questions 참고)
- 공유 링크로 타인의 워크스페이스 진입 시 조회만 허용, 로그인 유도 UI 표시

- 파일 업로드 크기 제한
- 협업 동시 접속자 수 제한
